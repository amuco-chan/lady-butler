# Lady's Butler — パーソナル執事AI MVP

日常タスク、予定、気分ログ、日記をひとつにまとめ、止まったときに「次の10分」を示す個人用Webアプリです。

## MVP仕様

- タスクの追加・編集・削除・完了、検索、状態フィルター、締切／優先度ソート
- 締切、優先度、残り進捗、所要時間に基づく「今日やること」の自動整理
- 設定、タスク、日記、気分ログをブラウザ内に自動保存
- 5段階の気分チェックインと一言メモ
- 短い日記、執事による振り返り、過去の日記一覧
- 気分に応じた今日のタスク量調整
- カレンダー画面で予定の追加・編集・削除
- Custom GPTから届いたタスク候補・予定候補を受信箱で確認して追加
- PC／スマートフォン対応
- スマートフォン用の下部タブと、ホーム画面へ追加できるPWA構成

## 技術構成

- React + TypeScript + Vite
- CSSによるレスポンシブUI
- Lucide React（アイコン）
- localStorage（個人用MVPのデータ保存）
- Vercel Functions（Custom GPT用の取り込みリンク生成）
- 執事らしい短い振り返り文のローカル生成

チャット機能と課題サポート機能はアプリから外しています。記録と提案は、タスク・予定・気分ログ・日記を中心に使います。Custom GPTからの連携は、まず受信箱に入り、ユーザーが確認してからタスクや予定に追加します。

## 画面一覧

1. ホーム：今日の最優先、予定時間、締切間近、最初の10分
2. タスク：一覧、検索、並び替え、登録、編集、削除、完了
3. カレンダー：予定一覧、予定追加、編集、削除、自然文からのGPT連携案内
4. 日記・気分：気分、できたこと、しんどかったこと、明日への持ち越し、AIコメント
5. 設定：呼び名、口調、厳しさ、通知頻度、GPT連携URL、データ削除

## Custom GPTとつなぐ

Custom GPTのActionsに、次のOpenAPI URLを登録します。

`https://lady-butler.vercel.app/gpt-action-openapi.json`

GPTが課題・買い物などをタスク候補として、面談・授業・バイト・約束など時間の決まったものを予定候補として送ると、アプリ取り込み用のURLが返ります。そのURLを開くとホームの「GPTから届いた候補」に入り、確認してからタスクまたは予定に追加できます。現時点では外部データベースを使わないため、勝手に直接保存せず、開いた端末のブラウザ内に取り込みます。

## データ設計

`Task`、`CalendarEvent`、`MoodLog`、`DiaryEntry`、`Settings` の型を `src/types.ts` に定義しています。Taskには締切、カテゴリ、優先度、進捗、推定時間、状態、メモ、作成・更新日時を保持します。CalendarEventには開始・終了、場所、メモを保持します。気分と日記も端末のlocalStorageに保存し、外部へ送信しません。

## 開発手順

1. 画面とデータ型を定義
2. 端末内の保存処理を追加
3. タスクCRUDと今日の優先順位ロジックを実装
4. 日記の振り返りを実装
5. レスポンシブUIを調整
6. TypeScriptビルドとブラウザ操作で検証

## 起動方法

```bash
npm install
npm run dev
```

動作検査は `npm test`、本番用ビルドは `npm run build` で実行できます。

表示された `http://localhost:5174` をブラウザで開きます。本番用ファイルは `npm run build` で生成できます。

## スマートフォンで使う

GitHub Pagesで公開すると、PCを起動していないときも次のURLから開けます。

`https://amuco-chan.github.io/lady-butler/`

iPhoneはSafariの共有メニューから「ホーム画面に追加」、Androidはブラウザメニューから「アプリをインストール」を選べます。

タスク・日記・気分ログは、開いた端末のブラウザ内に保存されます。PCとスマートフォン間の自動同期は、将来のデータベース連携で追加できます。

## 主なファイル

- `src/App.tsx` — 画面、タスク操作、フォーム
- `src/lib.ts` — 保存、優先順位、執事応答、GPT受信データ変換
- `src/types.ts` — データ型
- `src/styles.css` — UIデザインとスマートフォン対応
- `api/gpt-inbox.js` — Custom GPTから受け取った候補を取り込みリンクに変換
- `public/gpt-action-openapi.json` — Custom GPT Actions用のOpenAPI定義
